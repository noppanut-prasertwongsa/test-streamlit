อันนีี้ใช้ env ของ streamlit-env

conda activate streamlit-env


### **วิธีใช้งาน:**
1.  สร้างโฟลเดอร์สำหรับโปรเจกต์ของคุณ
2.  นำโค้ดส่วนแรกไปบันทึกเป็นไฟล์ชื่อ `train_model.py` ในโฟลเดอร์นั้น
3.  นำโค้ดส่วนที่สองไปบันทึกเป็นไฟล์ชื่อ `app.py` ในโฟลเดอร์เดียวกัน
4.  เปิด Terminal หรือ Command Prompt แล้วเข้าไปที่ Directory ของโปรเจกต์นั้น
5.  รันคำสั่งเพื่อสร้างและบันทึกโมเดลก่อน:
    ```bash
    python train_model.py
    ```
    เมื่อรันเสร็จ คุณจะเห็นไฟล์ใหม่ชื่อ `house_price_model.joblib` ในโฟลเดอร์ของคุณ
6.  หลังจากนั้น รันเว็บแอปด้วยคำสั่ง:
    ```bash
    streamlit run app.py
    ```
    เบราว์เซอร์ของคุณจะเปิดหน้าเว็บแอปขึ้นมาโดยอัตโนมัติ และคุณก็สามารถเริ่มใช้งานได้ทันที
